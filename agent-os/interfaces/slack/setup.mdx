## Get your URL

Slack needs a public HTTPS URL to deliver events. For local development, use ngrok:

```bash
ngrok http 7777
```

Copy the `https://` forwarding URL. For production, use your deployed URL.

## Create a Slack App

<Tabs>
<Tab title="Quick start">

1. Download the [manifest.json](https://github.com/agno-agi/agno/blob/main/libs/agno/agno/os/interfaces/slack/manifest.json)
2. Replace `https://your-url` with your ngrok URL
3. Go to [api.slack.com/apps](https://api.slack.com/apps) → **Create New App** → **From an app manifest**
4. Paste the manifest and click **Create**
5. Go to **Install App** → **Install to Workspace**
6. Copy the **Bot User OAuth Token** (`xoxb-...`)

</Tab>
<Tab title="Step by step">

### Create the app

1. Go to [api.slack.com/apps](https://api.slack.com/apps)
2. Click **Create New App**
3. Select **From an app manifest**
4. Select your workspace
5. Choose **JSON** and paste the manifest below

Replace `https://your-url` with your ngrok or production URL:

```json
{
    "display_information": {
        "name": "My Agent",
        "description": "An AI agent that lives in Slack.",
        "background_color": "#000000"
    },
    "features": {
        "app_home": {
            "home_tab_enabled": false,
            "messages_tab_enabled": true,
            "messages_tab_read_only_enabled": false
        },
        "bot_user": {
            "display_name": "My Agent",
            "always_online": true
        },
        "assistant_view": {
            "assistant_description": "Ask me anything.",
            "suggested_prompts": []
        }
    },
    "oauth_config": {
        "scopes": {
            "bot": [
                "app_mentions:read",
                "assistant:write",
                "channels:history",
                "channels:read",
                "chat:write",
                "files:read",
                "files:write",
                "groups:history",
                "groups:read",
                "im:history",
                "users:read",
                "users:read.email"
            ]
        }
    },
    "settings": {
        "event_subscriptions": {
            "request_url": "https://your-url/slack/events",
            "bot_events": [
                "app_mention",
                "assistant_thread_started",
                "message.im"
            ]
        },
        "org_deploy_enabled": false,
        "socket_mode_enabled": false,
        "token_rotation_enabled": false
    }
}
```

6. Click **Create**

The manifest configures scopes, events, and app home settings in one shot.

6. Click **Create**

### Install to workspace

1. Go to **Install App** in the sidebar
2. Click **Install to Workspace**
3. Click **Allow** to authorize
4. Copy the **Bot User OAuth Token** (`xoxb-...`)

</Tab>
</Tabs>

## Set Environment Variables

```bash
# Bot User OAuth Token (from Install App page)
export SLACK_TOKEN="xoxb-..."

# Signing Secret (Basic Information → App Credentials)
export SLACK_SIGNING_SECRET="..."
```

## Run

```python basic.py
from agno.agent import Agent
from agno.models.openai import OpenAIResponses
from agno.os.app import AgentOS
from agno.os.interfaces.slack import Slack

agent = Agent(name="Support Bot", model=OpenAIResponses(id="gpt-5.4"))

app = AgentOS(
    agents=[agent],
    interfaces=[Slack(agent=agent)],
).get_app()
```

```bash
python basic.py
```

DM the bot or @mention it in a channel to test.

## Verify

**Direct message**: Find your bot under **Apps** in the Slack sidebar and message it directly.

**In a channel**: Invite the bot first, then mention it:

```
/invite @MyAgent
@MyAgent what can you help me with?
```

Each thread maintains its own conversation. Follow-up messages in the same thread don't need to mention the bot again.

## Customizing the Manifest

### Minimal scopes

The manifest above includes scopes for file handling and user lookup. For a minimal bot that only responds to messages:

```json
"bot": [
    "app_mentions:read",
    "assistant:write",
    "channels:read",
    "chat:write",
    "im:history"
]
```

### Respond to all channel messages

Add `message.channels` and `message.groups` to bot_events, and set `reply_to_mentions_only=False` in your code:

```json
"bot_events": [
    "app_mention",
    "assistant_thread_started",
    "message.channels",
    "message.groups",
    "message.im"
]
```

```python
Slack(agent=agent, reply_to_mentions_only=False)
```

### Search tools

For `SlackTools.search_workspace()`, add these scopes:

```json
"bot": [
    "search:read.public",
    "search:read.files",
    "search:read.users"
]
```

See the [Reference](/agent-os/interfaces/slack/reference) for all scopes and their purposes.

<Warning>
ngrok URLs change each time you restart. For production, deploy behind a stable URL. See [deployment templates](/deploy/templates).
</Warning>
