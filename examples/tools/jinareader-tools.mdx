---

Enable Agno agents to convert any URL, PDF, or HTML into high-quality, LLM-friendly Markdown in real-time with Jina Reader.

```python

from agno.agent import Agent
from agno.tools.jina import JinaReaderTools

# ---------------------------------------------------------------------------
# Create Agent
# ---------------------------------------------------------------------------

agent = Agent(tools=[JinaReaderTools()])

# ---------------------------------------------------------------------------
# Run Agent
# ---------------------------------------------------------------------------
if __name__ == "__main__":
    agent.print_response("Summarize: https://github.com/agno-agi/agno")
```

## Run the Example
```bash
# Clone and setup repo
git clone https://github.com/agno-agi/agno.git
cd agno/cookbook/91_tools
```
