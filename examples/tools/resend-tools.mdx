title: "Resend"
description: "Use Resend tools with Agno agents."
---

Enable Agno agents to move beyond the chat interface and interact with users directly in their inbox for sending emails, batch delivery, scheduling and status tracking.

```python

from agno.agent import Agent
from agno.tools.resend import ResendTools

# ---------------------------------------------------------------------------
# Create Agent
# ---------------------------------------------------------------------------

from_email = "<enter_from_email>"
to_email = "<enter_to_email>"

agent = Agent(tools=[ResendTools(from_email=from_email)])

# ---------------------------------------------------------------------------
# Run Agent
# ---------------------------------------------------------------------------
if __name__ == "__main__":
    agent.print_response(f"Send an email to {to_email} greeting them with hello world")
```
