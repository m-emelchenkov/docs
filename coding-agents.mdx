---
title: Use Agno with Coding Agents
description: "Give your coding agent access to Agno documentation."
---

Two ways to get Agno's full documentation accessible to your coding agents.

## Option 1: Add the Docs as an MCP Server

Add [https://docs.agno.com/mcp](https://docs.agno.com/mcp) as an MCP server in your coding agent. The agent gets live access to the docs directly.

Works in any MCP-aware client: Claude Code, Cursor, Codex, Continue, and others.

### Claude Code example

Two ways to give claude code access to agno. Add it using the CLI:

```bash
claude mcp add agno-docs https://docs.agno.com/mcp
```

Or commit a `.mcp.json` to your repo so anyone cloning it gets the docs wired in automatically:

```json .mcp.json
{
  "mcpServers": {
    "agno-docs": {
      "type": "http",
      "url": "https://docs.agno.com/mcp"
    }
  }
}
```

Then in any session, the agent can search, read pages, and pull code samples on demand.

## Option 2: Index the docs

Add `https://docs.agno.com/llms-full.txt` as an indexed source. Your coding agent reads it like any other doc reference.

| Tool         | Where to add it                                                    |
| ------------ | ------------------------------------------------------------------ |
| **Cursor**   | Settings → Indexing & Docs → Add `https://docs.agno.com/llms-full.txt` |
| **VSCode**   | Add the URL as a doc source via your AI extension's settings.       |
| **Windsurf** | Add it to your indexed sources in the Cascade settings.             |

Re-index periodically as the docs change.
