## Overview

A workflow that processes meeting recordings, extracts action items, and automatically creates issues in Linear with assignees and due dates.

## Planned Features

- Meeting transcription processing
- Action item extraction with context
- Assignee detection from conversation
- Due date inference
- Linear issue creation
- Meeting summary generation
- Follow-up reminder scheduling

## Workflow Steps

```
1. Receive meeting transcript/recording
2. Extract action items with owner mentions
3. Map owners to Linear users
4. Create issues with descriptions
5. Set priorities and due dates
6. Generate meeting summary
```

## Use Cases

- Sprint planning meetings
- Client calls
- Team standups
- Project kickoffs
- One-on-ones
